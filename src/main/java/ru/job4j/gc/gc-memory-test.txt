Пустой объект без полей потребляет 16 байт памяти (на заголовок)

Потребляемая память для одного объекта new User(1, "N1"):
заголовок - 16
поле age - 4
поле name - 44 =
    16 (загаловок)
    + 4 (hash)
    + 4 (reference)
    + 16 (загаловок массива char[])
    + 2 * 2 (байт на символ)
итого: 64

Запуски:

для -Xmx3m -Xms2m:
1000 объектов - GC не вызывается
10000 объектов - GC вызывается
50000 объектов - GC вызывается, но падает OOM

для -Xmx6m -Xms4m:
50000 объектов - GC вызывается, но падает OOM

для -Xmx12m -Xms8m:
50000 объектов - GC вызывается, OOM больше не падает
100000 объектов - GC вызывается, но падает OOM
